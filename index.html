HTML

HTML setup

How to setup your HTML web page

Tag Name Description
<!DOCTYPE html> HTML doctype Defines the doctype of the document.
<html> HTML tag Represents the root of an HTML document and is the containter for all other HTML elements (except for the DOCTYPE-tag).
<head> Head element The head element is a container for metadata (data about data) and is placed between the html-tag and the body-tag.
<title> Document title Defines the title of the document. This must be text-only, and is shown in the browser's title bar or in the page's tab. Required in HTML documents.
<body> Document's body Defines the document's body. Contains all the content of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

CSS

CSS: margins and paddings

When and how to use them

Tag Name Description
Margin of an HTML element The margin property can be used to create space around elements, outside of any defined borders.
margin: auto; Margin property Margin is calculated by the browser.
margin: 30px; Specifies a margin in px (can also be set in cm, pt, etc.).
margin: 40%; Specifies a margin in % of the width of containing element.
margin: inherit; Specifies that the margin should be inherited from the parent element.
The padding of an element. The padding is used to create space around an elemnt's content, inside of any defined borders.
padding: 30px; Padding property Specifies a padding in px, pt, cm, etc.
padding: 40%; Specifies a padding in % of the width of the containing element.
padding: inherit; Specifies that the padding should be inherited from the parent element.
